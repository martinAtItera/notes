# Learning Resources

A curated list of learning resources for various topics.

## Programming

### Web Development
- [MDN Web Docs](https://developer.mozilla.org/) - Comprehensive web development documentation
- [freeCodeCamp](https://www.freecodecamp.org/) - Free coding bootcamp
- [Frontend Masters](https://frontendmasters.com/) - Expert-led video courses

### Computer Science
- [CS50](https://cs50.harvard.edu/) - Harvard's intro to computer science
- [MIT OpenCourseWare](https://ocw.mit.edu/) - Free MIT courses
- [Coursera](https://www.coursera.org/) - Online courses from top universities

## Books

### Software Development
- "Clean Code" by Robert C. Martin
- "The Pragmatic Programmer" by Hunt & Thomas
- "Design Patterns" by Gang of Four

### Software Architecture
- "Building Microservices" by Sam Newman
- "Domain-Driven Design" by Eric Evans

## Practice

- [LeetCode](https://leetcode.com/) - Coding interview practice
- [HackerRank](https://www.hackerrank.com/) - Programming challenges
- [Project Euler](https://projecteuler.net/) - Mathematical programming problems

## Stay Updated

- [Hacker News](https://news.ycombinator.com/) - Tech news and discussions
- [Dev.to](https://dev.to/) - Developer community
- [GitHub Trending](https://github.com/trending) - Trending repositories
