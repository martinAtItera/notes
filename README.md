# Notes Repository

A repository for different text and markdown files I want to share.

## Structure

This repository contains various notes organized by topic:

- **development/** - Development and programming notes
- **learning/** - Learning resources and study notes
- **projects/** - Project ideas and documentation

## Usage

Feel free to browse through the folders to find notes on different topics. All notes are in either markdown (`.md`) or plain text (`.txt`) format for easy reading.
